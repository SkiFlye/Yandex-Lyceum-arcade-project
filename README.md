Проект: Fate Deck: Карточный Рогалик
Срок сдачи: 30.01.2026

Краткое описание:
Карточный рогалик на Python + библиотека Arcade. Игрок исследует случайные подземелья, сражается с врагами через уникальную систему карт, собирает реликвии и улучшает персонажа.

Особенности:
4 специальные масти карт вместо стандартных
Случайная генерация подземелья
Пошаговые бои
Система прокачки и реликвий

Что делает игра:
1. Система карт
4 масти с разными эффектами:
1) Меч — урон врагу
2) Щит — блокирование урона
3) Зелье — усиление следующих карт
4) Чары — ослабление врага
Значения: 2-10 (у каждого свои множители эффектов)

2. Боевая система
По очереди: игрок → враг → игрок
Нажатие по карте → применение ее эффекта
Враг атакует автоматически
Победа при HP врага = 0

3. Подземелье
Сетка 5×5 комнат
Типы комнат:
1) Враг
2) Сокровище
3) Магазин
4) Событие
Перемещение возможно по соединенным комнатам

4. Прокачка
1) Уровни персонажа
2) Реликвии (обеспечивают пассивные бонусы)
3) Покупка улучшений в магазине

Как будет выглядить игра:
Экраны:
Главное меню — начало, выход
Карта подземелья — исследование комнат
Бой — враг сверху, карты игрока снизу
Магазин — покупка улучшений

Интерфейс:
Полоски HP игрока и врага
Рука из карт (3-5 штук)
Карта подземелья миниатюрой

Управление:
WASD — передвижение 
Мышь — выбор карт
Пробел — пропуск хода
R — сброс руки
ESC — меню
